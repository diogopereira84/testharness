The **Fedex_MarketplaceCheckout** module **enables**:

<br>

> **On the cart:** 



- The split by seller for 1P and 3P sellers.
- Adds Delivery Methods section for each seller.
- Change edit button function to add Navitor URL. (Currently mocked).

<br>

> **On the checkout:** 
- Adds some functionalities from Mirakl_FrontendDemo module v1.0.1 on checkout.
  - It requires a "Shipping Zone" configured in Mirakl and Magento Admin.
  ---
  <br>

- The split by seller for 1P and 3P within order summary.
- Adds Estimated Pickup Time section.
- Adds Marketplace Delivery Methods.
- Changes the style of all steps.
- Shows pickup and shipping data on the review and submit step for mixed carts (Currently mocked).

</br>

### **Module Toggle**
---

It **can be toggled** with the "Enable Marketplace {Feature}" 
option **on the admin panel**:

**->** Stores 

**->** (Settings) configuration 

**->** Fedex Internal Configuration Tab

**->** Features Toggle Configuration

- Enable Marketplace **Cart** (Yes / No)

- Enable Marketplace **Checkout** (Yes / No)
